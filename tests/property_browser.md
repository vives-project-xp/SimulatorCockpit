## FlightGear – Real-time values bekijken via de Property Browser

### Property Browser openen (tijdens simulatie)
1. Beweeg de muis naar de bovenkant van het scherm
2. Open **ontwikkeling**
3. Klik op **blader door interne eigenschappen**
4. Nu opent de property browser


## belangerijkste values route in property browser
speed = /instrumentation/airspeed-indicator
fuel level = /consumables/fuel/total-fuel-lbs£
altitude = /instrumentation/altimeter/indicated-altitude-f
direction = /instrumentation/heading-indicator/
heading = /orientation/heading-deg/
